What is HTML?

HTML stands for Hypertext Markup Language, and it is a mark up language to define the structure of web pages

It uses tags (also called elements) and attributes to describe the structure

The structure of the page is organized in tree-like structure, and the structure is called DOM, Document Object Model This becomes important later when we want to interact with these elements


How is the page organized?

All html pages begin with the tag 

<!DOCTYPE html>
. This denotes that this document is of html type

Further more, the pages are organized in the following way 


            <html>
                <head>

                </head>
                <body>

                </body>
            </html>

The Head

The head tag in html contains meta data (the data about itself) about the page. In the head tag, it contains information such as which encoding it is using and title for the page. This is also the place to include the link to external stylesheets and <style></style> tag.

The Body

This is where the content of the page goes to. Everything you see in the web browser's viewport is here.

Attributes

Attributes are fields that provides additional information of a tag. For example, anchor tag has href attribute to describe its destination, image tag has src attributes to denote the resource's location.

Global Attributes are attributes that are available to all elements. They include attributes such as class or id.

Classes and Id's are used to identify elements. Classes are used to identify a group of elements where as id's are used to identify a single (unique) element in the page
